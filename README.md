どこどこJP　Ruby ライブラリ
===
Rubyを使用して簡単にどこどこJP判定値にアクセスするためのクラスライブラリです。

利用方法
===
Wikiページを参照してください。

前提条件
===
* Ruby 1.8.x 以上　（Ruby 1.8.5 および Ruby 2.1.5 で動作確認をしました。）
